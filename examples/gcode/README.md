# GCode Editor Examples

This folder contains small G-code files demonstrating the syntax and features used by the `gcode_editor` component.

Files:
- `axis_examples.gcode` — X/Y/Z/E axis tokens and moves
- `parameter_examples.gcode` — F, S, T, and P parameters
- `comments_examples.gcode` — semicolon and parenthesis comments
- `variables_examples.gcode` — multi-letter variables and macro-like tokens
- `ocodes_examples.gcode` — O-code program markers and calls
- `numbers_examples.gcode` — integer, decimal, negative, exponential numbers
- `combined_examples.gcode` — a combined file exercising many features

Usage:
- Open these files externally in any editor or use them with the `examples/simple_editor.rs` example UI to visualize syntax highlighting.

Notes:
- These are small, focused examples meant for testing and demonstration. Feel free to add additional cases (G/M variations, edge cases, etc.).
